## GitHub User Activity

https://roadmap.sh/projects/github-user-activity

Run `py .\main.py <username> ` to see users events by type, e.g. `py .\main.py "kamranahmedse"`

Showing information about:
- WatchEvent
- PushEvent
- IssuesEvent
- IssueCommentEvent
- PullRequestEvent
- ForkEvent


[About GitHubAPI events](https://docs.github.com/en/enterprise-server@3.15/rest/using-the-rest-api/github-event-types?apiVersion=2022-11-28)
